# ProyectoBrandonGrupo1
How to play
Flujo básico de una partida
1.El jugador inicia la partida.
2.Se presenta el estado actual:
  Vida del jugador
  Vida del enemigo
  Opciones disponibles
3.En cada turno, el jugador puede elegir una acción:
  Atacar
  Defenderse
  Usar un ítem
  Ejecutar una habilidad especial (si está disponible)
4.El enemigo responde según sus reglas.
5.El turno se repite hasta que alguien gane o pierda.

Ejemplo de partida
Inicio
Jugador: 100 HP
Enemigo: 80 HP
Turno 1
El jugador elige Atacar → Daño: 15
Enemigo queda con 65 HP
El enemigo contraataca → Daño: 10
Jugador queda con 90 HP
Turno 2
El jugador usa un ítem de curación → +20 HP
Jugador queda con 110 HP (máximo permitido aplicado si existe)
El enemigo ataca → Daño: 12
Turno 3
El jugador usa una habilidad especial
El enemigo queda con 0 HP
Resultado: Victoria del jugador.

Estadísticas
Vida (HP): permite margen de error, evitando que una mala decisión termine la partida de inmediato.
Daño de ataque: calculado para que los combates duren varios turnos, fomentando la estrategia.
Habilidades especiales: más poderosas, pero limitadas (uso, cooldown o condición).
Ítems: ayudan a recuperarse, pero no sustituyen una buena toma de decisiones.

Reglas clave
El enemigo responde siempre, evitando turnos “gratis”.
El jugador debe elegir entre riesgo inmediato y beneficio a largo plazo.
El progreso se basa en gestión de recursos, no solo en atacar.
